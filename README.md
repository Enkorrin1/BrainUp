# BrainUp

Образовательный продукт для тренировки логики, математики, внимания и мышления в формате коротких ежедневных уроков, прогресса по навыкам, серий занятий и мягкой игровой мотивации.

Идея проекта: взять за основу существующий проект `D:\Project\LogicLike`, сохранить его предметную ценность и мобильный Flutter-стек, но изменить форму обучения на Duolingo-style: линейная карта, короткие уроки, серия дней, сердца, XP, ежедневная цель, повторение ошибок и более сильная привычка возвращаться.

## Продуктовая формула

Пользователь заходит каждый день на 5-10 минут, проходит 1-3 коротких урока на линейной карте, решает оригинальные логические задачи с моментальной обратной связью, укрепляет слабые навыки через повторение и видит понятный прогресс как в языковых приложениях.

## Целевая аудитория

- Дети 6-12 лет, которым нужно развивать логику и математическое мышление.
- Родители, которые хотят видеть регулярный прогресс ребенка.
- Учителя и репетиторы, которым нужен структурированный тренажер.
- В перспективе: подростки и взрослые, которым интересны логические задачи как ежедневная тренировка мозга.

## Первый релиз

Первый релиз должен доказать главный цикл:

1. Пользователь открывает Duolingo-style карту обучения.
2. Проходит короткий урок из нескольких задач.
3. Получает мгновенную обратную связь.
4. Видит прогресс, опыт, серию дней и следующий урок.
5. Возвращается завтра, потому что прогресс понятен и приятно продолжать.
