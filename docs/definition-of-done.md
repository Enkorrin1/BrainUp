# Definition of Done

## Для задания

Задание считается готовым, если:

- есть понятное условие;
- указан `PuzzleType`;
- указан `SkillTag`;
- указан правильный ответ;
- есть объяснение;
- есть hint key;
- есть localization keys для видимого текста;
- задача проверена вручную;
- задача не допускает двусмысленного ответа.

## Для урока

Урок считается готовым, если:

- содержит 5-10 задач;
- связан с конкретным узлом карты;
- имеет понятную цель;
- проходится за 5-10 минут;
- корректно сохраняет результат;
- показывает финальный экран;
- начисляет XP;
- обновляет streak и hearts;
- ошибочные задания попадают в mistake review.

## Для карты

Карта считается готовой, если:

- есть текущий узел;
- есть completed/current/locked состояния;
- следующий урок очевиден;
- прогресс сохраняется;
- завершение урока открывает следующий узел;
- повторительные и наградные узлы не ломают основной маршрут.

## Для MVP-релиза

MVP считается готовым, если:

- пользователь может пройти полный учебный цикл;
- главный экран является Duolingo-style картой;
- прогресс сохраняется локально;
- есть минимум 3 навыка;
- есть минимум 20 уроков;
- есть минимум 100 задач;
- работает streak;
- работает XP;
- работают hearts;
- работает mistake review;
- есть родительская сводка прогресса;
- все видимые строки проходят через Flutter l10n;
- `flutter analyze` проходит;
- `flutter test` проходит;
- основной сценарий протестирован вручную на Android emulator;
- iOS-риски описаны для проверки на macOS/Xcode;
- нет критических ошибок, блокирующих обучение.
