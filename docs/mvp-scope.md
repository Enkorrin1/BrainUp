# MVP Scope

## Цель MVP

Проверить, что существующая оригинальная логическая база задач лучше работает как Duolingo-style обучение: ребенок видит карту, один следующий урок, серию, XP, сердца и повторение ошибок.

## Входит в MVP

### Обучение

- 3-5 тематических навыков.
- 20-30 уроков.
- 100-150 задач.
- 5-10 задач в одном уроке.
- Простое объяснение после ошибки.
- Повторение ошибочных задач.
- Прогресс по линейной карте.

### Продукт

- Главная Duolingo-style карта обучения вместо learning hub как первого экрана.
- Экран урока.
- Экран результата.
- Прогресс по навыкам.
- XP.
- Streak.
- Hearts.
- Daily goal.
- Mistake review.
- Простая родительская сводка прогресса.

### Технологии

- Flutter/Dart.
- Android и iOS.
- `SharedPreferences` для локального хранения.
- ARB EN/RU локализация.
- `flutter_svg` для пазловых ассетов.
- `flutter_test` для доменных и widget-тестов.

### Контент

Первые навыки:

- закономерности;
- сравнение и классификация;
- последовательности;
- простая арифметическая логика;
- внимание и исключение лишнего;
- память;
- пространственные задачи.

### Администрирование

На MVP контент можно хранить в Dart seed-каталоге, как в исходном `LogicLike`, или подготовить JSON-структуру для будущего импорта. Полноценная CMS не обязательна.

## Не входит в MVP

- оплата;
- AI-генерация заданий;
- социальные функции;
- сложные рейтинги;
- отдельный backend;
- кабинет учителя;
- полноценная облачная синхронизация;
- продвинутый адаптивный алгоритм.

## MVP-гипотезы

1. Короткий урок повышает вероятность завершения сессии.
2. Карта помогает ребенку понимать следующий шаг лучше, чем каталог.
3. Streak и daily goal повышают возвращаемость.
4. Hearts дают напряжение, но не должны демотивировать.
5. Повторение ошибок улучшает точность решения.
6. Родительская сводка повышает доверие к продукту.

## MVP-метрики

- завершение первого урока;
- завершение 3 уроков;
- возвращение на следующий день;
- среднее количество уроков за неделю;
- точность по навыкам;
- доля пользователей, которые повторяют ошибки;
- количество потерянных hearts;
- streak length;
- время до завершения урока.
